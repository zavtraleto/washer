# Washer Game

A TypeScript + React + Phaser 3 game built with Vite.

## Quick Start

```bash
# Install dependencies
npm install

# Run dev server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Technology Stack

- **Phaser 3.80** — Game runtime with WebGL rendering
- **Vite** — Fast HMR and builds
- **TypeScript** — Strict mode with enhanced checks
- **ESLint + Prettier** — Code quality

### Path Aliases

```typescript
import { App } from '@app/App';
import { DemoScene } from '@game/phaser/scenes/DemoScene';
```

## Setup

Install dependencies:

```bash
npm install
```

## Development

Run the development server:

```bash
npm run dev
```

Open your browser to the URL shown in the terminal (typically `http://localhost:5173`).

## Build

Build for production:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```

## Linting

Run ESLint:

```bash
npm run lint
```
